Vim adalah teks editor berbasis terminal yang kuat dan sangat konfigurabel. Berikut adalah beberapa perintah dasar yang sering digunakan dalam Vim:

1. Mode Normal (Normal Mode):

i: Masuk ke Mode Insert (untuk menulis atau mengedit teks).
x: Menghapus karakter di bawah kursor.
dd: Menghapus baris saat ini.
yy: Menyalin baris saat ini.
p: Menempelkan teks yang disalin setelah kursor.
u: Undo (membatalkan perubahan).
Ctrl + r: Redo (mengulangi perubahan yang dibatalkan).

2. Mode Insert (Insert Mode):

Esc: Keluar dari Mode Insert dan kembali ke Mode Normal.
:w: Menyimpan perubahan ke dalam file (write).
:q: Keluar dari Vim (quit).
:wq: Menyimpan perubahan dan keluar dari Vim.

3. Mode Visual (Visual Mode):

v: Memulai mode visual karakter, yang memungkinkan Anda untuk memilih teks karakter per karakter.
V: Memulai mode visual baris, yang memungkinkan Anda untuk memilih baris secara keseluruhan.

4. Navigasi (Navigation):

h, j, k, l: Menggerakkan kursor ke kiri, bawah, atas, dan kanan, secara berturut-turut.
w: Menggerakkan kursor ke kata berikutnya.
b: Menggerakkan kursor ke kata sebelumnya.
G: Menggerakkan kursor ke akhir file.
gg: Menggerakkan kursor ke awal file.

5. Pencarian (Search):

/kata_kunci: Mencari kata_kunci ke depan.
?kata_kunci: Mencari kata_kunci ke belakang.
n: Melompat ke kemunculan berikutnya dari kata_kunci.
N: Melompat ke kemunculan sebelumnya dari kata_kunci.

6. Editing (Pengeditan):

:s/kata_lama/kata_baru/g: Mengganti semua kemunculan kata_lama dengan kata_baru dalam baris saat ini.
:s/kata_lama/kata_baru/gc: Sama seperti di atas, tetapi dengan konfirmasi.

7. File (Berkas):

:e nama_file: Membuka file dengan nama nama_file.
:sp nama_file: Membuka file dalam jendela horizontal.
:vsp nama_file: Membuka file dalam jendela vertikal.

8. Jendela (Window):

Ctrl + w + w: Beralih antara jendela.
Ctrl + w + s: Membagi jendela horizontal.
Ctrl + w + v: Membagi jendela vertikal.

9. Baris Perintah (Command Line):

:: Masuk ke Mode Baris Perintah untuk mengeksekusi perintah Vim.
Ini adalah beberapa perintah dasar yang digunakan dalam Vim. Vim memiliki banyak fitur dan perintah lainnya yang dapat digunakan untuk mengedit teks dengan efisien. Anda dapat mempelajari lebih lanjut tentang Vim dengan membaca dokumen bantuan internal dengan perintah :help.
